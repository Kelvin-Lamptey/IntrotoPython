## 🛠️ Required Software

### 1. 🐍 Python
Python is the programming language we will be learning. You need to install the Python interpreter so your computer can run Python code.
* **Download:** [Python Official Downloads](https://www.python.org/downloads/)
* *Note: Make sure to check the box that says **"Add Python to PATH"** during installation! (Windows users)*

### 2. 💻 Visual Studio Code (VS Code)
VS Code is our code editor (IDE). It's where we will write, edit, and run our code. It's lightweight, free, and incredibly powerful.
* **Download:** [Download VS Code](https://code.visualstudio.com/Download)
* *Recommended Extension:* Once installed, open VS Code, go to the Extensions tab (`Ctrl+Shift+X` or `Cmd+Shift+X`), and install the official **Python extension** by Microsoft.

### 3. 🌿 Git
Git is a version control system. It helps us track changes to our code, save our progress, and collaborate with others. It's an industry-standard tool you'll use throughout your career.
* **Download:** [Download Git](https://git-scm.com/downloads)
* *Note: The default installation settings are perfectly fine for this course.*

---

## 🎯 Next Steps

Once you have installed all three tools:
1. Open **VS Code**.
2. Open the terminal inside VS Code (`Terminal -> New Terminal`).
3. Type `python --version` (or `python3 --version` on Mac) to verify Python is installed!
4. Type `git --version` to verify Git is installed!

If you encounter any issues during the installation process, don't worry, notify us on the next class, and we will help you get sorted.

Happy Coding! 🎉
